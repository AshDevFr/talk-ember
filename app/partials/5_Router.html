Router

It translates the URL into a series of nested templates with their model

In Ember.js each State is represented by a URL

Defining Your Routes

app/router.js 


   Router.map(function() {
      this.route('todos', {path : '/'});
   });
      

   Router.map(function() {
      this.route('todos', {path : '/'}, function() {
         this.route('edit');
      });
   });

Dynamic segments

The route handler converts a URL into a model

app/routes/todos.js


   export default Ember.Route.extend({
      model () {
         return this.store.findAll('todo');
      }
   });

Loading / Error Substates

Ember provides a default implementation of the loading / error process

Routes 

   todos.edit.loading
