Components

Ember components are based on of the W3C Web Components specification

Defining a Component

app/templates/components/todo-item.hbs


   <h1 {{action "someAction"}}>{{title}}
   
{{desc}}

app/components/todo-item.js

   export default Ember.Component.extend({
      tagName : 'li',
      actions : {
         // ...
      }
   });

Using a Component

app/templates/todos.hbs

    
   {{#each model as |todo|}}
      {{todo-item todo=todo}}
   {{/each}}

Wrapping Content in a Component

app/templates/components/todo-item.hbs


   <h1 {{action "someAction"}}>{{title}}
   
{{desc}}

   
{{yield}}

app/templates/todos.hbs

    
   {{#each model as |todo|}}
      {{todo-item todo=todo}}
         
      {{/todo-item}}
   {{/each}}
